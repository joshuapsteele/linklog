About LinkLog

LinkLog is a personal link blog by Joshua Steele. It's a running log of interesting articles, tools, and resources I've come across, usually with a short note about why they caught my attention.

The site is built with Go and SQLite, self-hosted on a DigitalOcean droplet. New links are posted from my phone using the Drafts app. The source code and setup details are available in the GitHub repository.
